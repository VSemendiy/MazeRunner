# MazeRunner
In this project, I wrote a program that generates mazes and looks for a way out.

The program provides a menu with five options:
```
1. Generate a new maze.
2. Load a maze.
3. Save the maze.
4. Display the maze.
5. Find the escape.
0. Exit.
```
After a maze is generated or loaded from a file, it becomes the current maze that can be saved or displayed.    
If there is no current maze (generated or loaded), a user cannot see the options save, display the maze and find the escape. If a user chooses an option that requires a file, he must enter a path to the file.    

The program outputs the maze to the user only in two scenarios:
- after generating a maze
- after choosing an option `display the maze`
